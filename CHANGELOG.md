# DashHub 版本记录 (Changelog)

本文档用于记录 DashHub 项目的所有主要版本发布、功能更新、问题修复和优化说明。

## [v1.3.0] - 2026-03-16
### 🚀 新功能 (Features)
- **Dashtro SSO 互操作登录 (方案A)**：点击“访问项目大厅”按鈕时，如果项目接入了 Dashtro 且用户已登录，将自动将 DashHub JWT Token 附加进跳转 URL
- **创建者主页** `/creator/:userId`：点击项目创建者即可进入展示该用户创建的全部在架项目
- **语言软化**：全局广播才的赤裸金融词汇（分账/收益等），替换为“协作规模/活跃度”等技术中性表述

## [v1.2.0] - 2026-03-16
### 🚀 新功能 (Features)
- **庭非排行榜系统** `/leaderboard`：包含平台实力排行榜 + 本周新星榜双 Tab
- **首页改版**：分账价値最高项目的热点入口取代原来的广泛展示记录小板
- **Dashtro 连接准备**：创建与编辑项目表单内嵌入 Dashtro 连接推荐卡咯、开关和内容文本区
- **项目入口展示**：项目详情天首屏处放出在显著位置的“访问项目大厅”主入口按鈕
- **合作模式展示**：项目详情页新增“核心合作形式”模块，项目如接入 Dashtro 则展示协议明细

### 🔧 优化 (Improvements)
- “流量主”统一升级更名为“流量厂牌”
- 项目详情页大面积商业气质重构，采用黑色克制线条 + 大留白商业名片风格

---

## [v1.1.0] - 当前版本
### 🚀 核心功能 (Core Features)
- **项目管理体系**：实现项目的创建、更新、展示及删除。
- **项目详情展示**：支持图文并茂、多图片与多视频资源上传的高级文件系统。
- **现代化页面设计**：响应式的 UI 界面，提供完善的页脚、全局导航与跨设备兼容性。
- **搜索与发现**：基于全局项目数据的历史搜索引擎、热搜推荐体系。
- **用户体系重构**：完善的账户系统，支持个人设置管理与修改，提升用户体验。

### 🔧 优化 (Improvements)
- **前端上传组件重构**：从基础的表单全面升级为支持拖拽、进度条、多图批量处理的高级媒体组件（在新建和编辑项目等关键链路中均已实装）。
- **组件抽取与复用**：系统级别的 Navbar 及 Footer 引入，减少代码冗余并修复报错。
- **交互回馈加强**：关键链路按钮（保存、复制社交方式、删除等）提供实时的异步状态处理 (`v-loading`) 与路由智能回退。

### 🐛 修复 (Bug Fixes)
- 彻底修复 `Search.vue` 点击返回时状态丢失及参数不匹配导致 500 的问题。
- 修复 `Profile.vue` 部分数据库冗余字符串导致的后端聚合金额计算出现 `NaN` 乱码错误。
- 修复 `EditProject.vue` 前后端传参方式导致的 JSON 数据流保存引发服务器 500 死锁。
- 补齐了全局主要 Logo（顶导、底导及登录注册页）缺失的返回主页路由热链问题。
